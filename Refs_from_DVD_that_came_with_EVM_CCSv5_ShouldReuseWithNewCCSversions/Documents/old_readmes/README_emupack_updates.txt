The Emulation service release updates can be installed on the top of the specified CCS baseline release. 
Update Installation Steps
1. Install CCS
2. Copy file TIXDS2XXUSB_onboard_Connection.xml to the directory your_CCS_install_dir/ccsv5/ccs_base/common/targetdb/connections
3. Make sure the CCS is closed
4. Make sure the emulation is not connected
5. install ti_emupack_5.0.838.0
6. Install ti_emupack_keystone1_1.0.6.0 for the appropriate platforms
